# Compodoc 

Generate static documentation.

```cmd
npm run compodoc
```
Generate documentation not using the serve.

```cmd
npm run compodoc:g
```

Generate documentation and serve on `http://localhost:4200`.

```cmd
npm run compodoc:s
```

Generate documentation, serve on `http://localhost:4200` and watch changes.

```cmd
npm run compodoc:w
```
